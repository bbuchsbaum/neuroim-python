# Porting and Parity Notes

This folder holds historical and current notes from the `neuroim2` to
`neuroim` port. These files are useful for maintainers, but they are not part
of the public first-read package surface.

## Current References

- `PARITY_COMPLETION_AUDIT.md` records the latest parity audit and verification
  status.
- `FEATURE_PARITY.md` tracks the broad R-to-Python API/class surface.
- `MIGRATING_FROM_R.md` is the user-facing migration guide draft.
- `R_PYTHON_MAPPING.md` and `R_to_Python_test_mapping.md` preserve mapping
  notes from the original translation work.
- `PERFORMANCE_ISSUES.md` and `TRANSLATION_STRATEGY.md` are maintainer notes
  for future hardening work.
